# Amazon Alexa OAuth2

**Phase 1: OAuth2 Authentication Foundation**

Secure OAuth2 authentication for Amazon Alexa using PKCE (RFC 7636).

## Current Features (v0.1.0)
- OAuth2 with PKCE security
- Automatic token refresh
- Multi-account support

## Roadmap
- **v0.2**: Device discovery and switch control
- **v0.3**: Full device entity support (lights, climate, sensors, media players)
- **v1.0**: Automation triggers and advanced features

**Work in Progress** - Phase 1 of 4
